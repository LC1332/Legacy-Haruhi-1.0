# 项目介绍


## 前端技术栈
1. Vue
2. ElementUI
3. axios
4. vue-router
5. Vuex
6. WebSocket
7. vue-cli4

## Project setup
```
npm install
```

## 环境配置
指定包下载
```
#安装element-ui
npm i element-ui -S  
#安装axios
npm install axios
#安装vuex
npm install vuex --save
#安装font-awesome
npm install --save font-awesome
#安装sass
npm install sass-loader --save-dev
cnpm install node-sass --save-dev
```

## Compiles and hot-reloads for development 运行项目
```
npm run serve
```

## Compiles and minifies for production打包项目
```
npm run build
```
